¿Cuáles cree que son las ventajas y desventajas de utilizar Microdata?
Ventaja:
- se sirve de html
- da formato
Desventaja:
- puede estar disperso por html y dificil encontrarlo
- mezcla html y estilo con datos, limitando al programador

¿Cuáles cree que son las ventajas y desventajas de utilizar JSON-LD?
Ventaja:
- esta en json, es muy procesable
- separa datos de html
Desventaja:
- hace más pesado el archivo html
- no es algo muy usado todavia
